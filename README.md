Instructions
=============

Install the `AgeingError` package from this github site as follows:

```r
# using {remotes}
remotes::install_github("pfmc-assessments/AgeingError")
# using {pak}
pak::pak("pfmc-assessments/AgeingError")

# Load package
library(AgeingError)

```

See the vignette for example use.
