# Read the ageing error specifications

Read the ageing error specifications

## Usage

``` r
load_specs(SpecsFile = "data.spc", DataSpecs, verbose = FALSE)
```

## Arguments

- SpecsFile:

  Filename for input specifications.

- DataSpecs:

  The output from load_data()

- verbose:

  Return messages to the console (TRUE/FALSE)
